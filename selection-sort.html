Ordenamiento Selección (Selection-sort)

Funcionamiento

La forma de funcionamiento del algoritmo es la siguiente:


1. Se inicia recorriendo el arreglo desde la posición a[0], tomando el valor contenido en esta posición y comparándolo con cada uno de los ítems del arreglo hasta llegar a la posición a[n-1], durante esta primera iteración lo que se busca es encontrar cual de los valores contenidos en los elementos del arreglo es el menor, si el valor contenido en a[1] es menor al que esta en a[0], este es intercambiado y así se continua comparando el resto de elementos con el que se encuentra en a[0], se repite este proceso hasta llegar la posición a[n-1].


2. En la segunda iteración se repite el proceso anterior pero comenzando en la posición a[1] hasta llegar a a[n-1].


3. Se continúa repitiendo el proceso 1 en los elementos restantes del arreglo.


Este algoritmo es una versión mejorada del algoritmo burbuja, y a pesar de que maneja el orden de complejidad o(n²), es más rápido en comparación al algoritmo burbuja, debido a que se reducen notoriamente el numero de interacciones que realiza.

void swap(int *xp, int *yp)
{
    int temp = *xp;
    *xp = *yp;
    *yp = temp;
}
 
void selectionSort(int arr[], int n)
{
    int i, j, min_idx;
 
    // One by one move boundary of unsorted subarray
    for (i = 0; i < n-1; i++)
    {
        // Find the minimum element in unsorted array
        min_idx = i;
        for (j = i+1; j < n; j++)
          if (arr[j] < arr[min_idx])
            min_idx = j;
 
        // Swap the found minimum element with the first element
        swap(&arr[min_idx], &arr[i]);
    }
}

{
    int n = arr.length;
 
    // One by one move boundary of unsorted subarray
    for (int i = 0; i < n-1; i++)
    {
      // Find the minimum element in unsorted array
      int min_idx = i;
      for (int j = i+1; j < n; j++)
        if (arr[j] < arr[min_idx])
            min_idx = j;
 
      // Swap the found minimum element with the first
      // element
      int temp = arr[min_idx];
      arr[min_idx] = arr[i];
      arr[i] = temp;
    }
}

Sub Selectionsort(list As Variant, Optional min As Integer = -2, Optional max As Integer = -2)
Dim i As Integer
Dim j As Integer
Dim best_value As Variant 'Long
Dim best_j As Integer

    If min = -2 Then min = LBound(list)
    If max = -2 Then max = UBound(list)

    For i = min To max - 1
        best_value = list(i)
        best_j = i
        For j = i + 1 To max
            If list(j) < best_value Then
                best_value = list(j)
                best_j = j
            End If
        Next j
        list(best_j) = list(i)
        list(i) = best_value
    Next i
End Sub

Ejemplo

Lista x = [5,3,1,2,4]

Se inicia el proceso comparando el elemento de la posición a[0] con el de la posición a[1], al ver que 3 es menor que 5 se realiza en intercambio.

x = [3,5,1,2,4]

se continua comparando la posición a[0] con la posición a[2], como 1 es menor que 3 se realiza el intercambio.

x = [1,3,5,2,4]

se continua comparando la posición a[0] con la posición a[3], como 1 es menor que dos se deja tal y como está el arreglo.

x = [1,3,5,2,4]

se continua comparando la posición a[0] con la posición a[3], como 1 es menor que dos se deja tal y como está el arreglo. Aquí termina la primera iteración del arreglo. Seguido a esto se inicia con la segunda iteración en la cual se compara el valor contenido en a[2] con el de a[3], como 3 es menor que 5, se deja tal y como esta, y se continua comparando a[2] con el de a[4], al ver que 2 es menor que 3 se realiza el intercambio de elementos.

x = [1,2,5,3,4]

se continua comparando la posición a[1] con la posición a[4], como 1 es menor que dos se deja tal y como está el arreglo. Aquí termina la segunda iteración del arreglo. Se inicia con la tercera iteración y se compara el valor contenido en a[2] con el de a[3], al ver que 3 es menor que 5 se realiza el intercambio de elementos.

x = [1,2,3,5,4]

se continua comparando la posición a[2] con la posición a[4], como 1 es menor que dos se deja tal y como está el arreglo. Aquí termina la tercera iteración del arreglo. Se inicia con la cuarta iteración, en donde cual se compara el valor contenido en a[3] con el de a[4], al ver que 4 es menor que 5 se realiza el intercambio de elementos.

x = [1,2,3,4,5]

Con esto se terminan las iteraciones del algoritmo quedando la lista ordenada.
