Ordenamiento Burbuja (Bubble-sort)

Historia

El algoritmo es mencionado en varios artículos entre los años 1950 y 1962, se le ha mencionado con otros nombres en estos artículos pasando por Exchange sort, termino usado por la asociación para la maquinaria en la computación (ACM), en 1962 es utilizado por primera vez el término “Bubble Sort” por Iverson, a pesar de esto el algoritmo fue oficialmente publicado en la ACM en el año de 1975. Se volvió un algoritmo muy popular dentro de las personas que están iniciando en el campo de la programación debido a su facilidad para entenderlo y escribirlo en el lenguaje de preferencia.

Funcionamiento

La manera en que funciona este algoritmo es realizando un recorrido a una lista, comparando cada elemento de la lista con el siguiente, para así determinar cuál es el menor y cuál es el mayor, con esto el algoritmo realiza el intercambio de los dos ítems, es necesario que el algoritmo haga varios recorridos a la lista para poder realizar el ordenamiento de todos los ítems de la lista. Debido a que tiene un orden de complejidad cuadrático O(n²) hace que sea un algoritmo muy costoso para su uso, debido a que depende directamente del número, ya que entre más ítems tenga más comparaciones debe de hacer y la cantidad de recorridos que tendría que hacer serian n-1 cada vez.

Implementación

Para el algoritmo bubble sort podemos encontrar las siguientes implementaciones, dependiendo del lenguaje de programación, a continuación, se mostrará la implementación en los lenguajes de programación más usados.

C++

                        
void swap(int *xp, int *yp)
{
  int temp = *xp;
  *xp = *yp;
  *yp = temp;
}

void bubbleSort(int arr[], int n)
{
  int i, j;
  for (i = 0; i < n-1; i++)
    // Last i elements are already in place
    for (j = 0; j < n-i-1; j++)
      if (arr[j] > arr[j+1])
        swap(&arr[j], &arr[j+1]);
}

JAVA

                        
void bubbleSort(int arr[])
  {
     int n = arr.length;
     for (int i = 0; i < n-1; i++)
        for (int j = 0; j < n-i-1; j++)
           if (arr[j] > arr[j+1])
           {
              int temp = arr[j];
              arr[j] = arr[j+1];
               arr[j+1] = temp;
           }
  }
