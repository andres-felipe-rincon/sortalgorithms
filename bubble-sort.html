Ordenamiento Burbuja (Bubble-sort)

Historia

El algoritmo es mencionado en varios artículos entre los años 1950 y 1962, se le ha mencionado con otros nombres en estos artículos pasando por Exchange sort, termino usado por la asociación para la maquinaria en la computación (ACM), en 1962 es utilizado por primera vez el término “Bubble Sort” por Iverson, a pesar de esto el algoritmo fue oficialmente publicado en la ACM en el año de 1975. Se volvió un algoritmo muy popular dentro de las personas que están iniciando en el campo de la programación debido a su facilidad para entenderlo y escribirlo en el lenguaje de preferencia.

Funcionamiento

La manera en que funciona este algoritmo es realizando un recorrido a una lista, comparando cada elemento de la lista con el siguiente, para así determinar cuál es el menor y cuál es el mayor, con esto el algoritmo realiza el intercambio de los dos ítems, es necesario que el algoritmo haga varios recorridos a la lista para poder realizar el ordenamiento de todos los ítems de la lista. Debido a que tiene un orden de complejidad cuadrático O(n²) hace que sea un algoritmo muy costoso para su uso, debido a que depende directamente del número, ya que entre más ítems tenga más comparaciones debe de hacer y la cantidad de recorridos que tendría que hacer serian n-1 cada vez.

void swap(int *xp, int *yp)
{
  int temp = *xp;
  *xp = *yp;
  *yp = temp;
}

void bubbleSort(int arr[], int n)
{
  int i, j;
  for (i = 0; i < n-1; i++)
    // Last i elements are already in place
    for (j = 0; j < n-i-1; j++)
      if (arr[j] > arr[j+1])
        swap(&arr[j], &arr[j+1]);
}

static void bubbleSort(int []arr)
{
    int n = arr.Length;
    for (int i = 0; i < n - 1; i++)
    {
        for (int j = 0; j < n - i - 1; j++)
        {
            if (arr[j] > arr[j + 1])
            {
                // swap temp and arr[i]
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
       }
    }
}
                      
                    
                      
function bubbleSort ($items) {
   $size = count($items);
   for ($i=0; $i<$size; $i++) {
        for ($j=0; $j<$size-1-$i; $j++) {
             if ($items[$j+1] < $items[$j]) {
                 arraySwap($items, $j, $j+1);
             }
        }
   }
   return $items;
}

function arraySwap (&$arr, $index1, $index2) {
   list($arr[$index1], $arr[$index2]) = array($arr[$index2], $arr[$index1]);
}

                      
                    
                      
Public Sub BubbleSort(ByRef ArrayIn() As Long)
  Dim i, j    As Integer
  Dim t       As Long
  For i = UBound(ArrayIn) To 0 Step -1
    For j = 0 To i - 1
        If ArrayIn(j) > ArrayIn(j + 1) Then
            Call swap(ArrayIn(j), ArrayIn(j + 1))
        End If
    Next j
  Next i
End Sub

Private Sub swap(ByRef data1 As Long, ByRef data2 As Long)
  Dim temp As Long
  temp = data1
  data1 = data2
  data2 = temp
End Sub

Ejemplo

Se inicia comparando el primer elemento con el segundo, 5 es mayor a 3, con esto realizamos el intercambio de los dos elementos estableciendo el 3 como el menor y el 5 como el mayor, quedando el arreglo de la siguiente manera:

x = [3,5,1,2,4]

Se vuelve a realizar otro recorrido en el algoritmo, comparando si el 5 es mayor a 1, y realizando el intercambio.

x = [3,1,2,5,4]

Se vuelve a realizar otro recorrido en el algoritmo, comparando si el 5 es mayor a 4, y realizando el intercambio.

x = [3,1,2,4,5]

Se vuelve a realizar otro recorrido en el algoritmo, comparando si el 3 es mayor a 1, y realizando el intercambio.

x = [1,3,2,4,5]
