Ordenamiento por inserción (insert-sort)

Historia

La primera aparición en los textos de este método fue en el año de 1946 John Mauchly lo menciono por primera vez en un artículo acerca de ordenamiento computacional.

Funcionamiento

La forma de funcionamiento de este algoritmo es la siguiente: Debemos de tener en cuenta que en la lista es subdividida en 2 partes, una parte ordenada y una desordenada. La parte ordenada comienza siempre con la posición a[0].

  1. Se inicia el recorrido de la lista ordenada y se agrega la posición a[1], este ítem es comparado con la posición a[0], si a[1] es menor que a[0], a[0] es desplazado a la derecha y el valor contenido en a[1] es asignado a a[0], si a[1] es mayor continua en esta posición.
  2. En la siguiente iteración se agrega el siguiente elemento de la lista a la lista ordenada, y se realiza de nuevo la comparación entre los elementos contenidos en la lista, si es necesario realizar los intercambios necesarios para que la lista quede ordenada.
  3. Se continúan las iteraciones repitiendo lo realizado en el paso 2 hasta llega a la posición a[n-1], aquí terminan las iteraciones y se tiene como resultado la lista ordenada completamente.

                      void insertionSort (int a[], int n)
                      {
                        int i, j, aux;
                        for (i = 1; i < n; i++)
                        {
                          /* indice j es para explorar la sublista a[i-1]..a[0] buscando la posicion
                          correcta del elemento destino */
                          j = i;
                          aux = a[i];
                          // se localiza el punto de inserción explorando hacia abajo
                          while (j > 0 && aux < a[j-1])
                          {
                            // desplazar elementos hacia arriba para hacer espacio
                            a[j] = a[j-1];
                            j--;
                            }
                            a[j] = aux;
                          }
                        }
                      }                      
                      
                  
                    
                      void sort(int arr[])
                      {
                          int n = arr.length;
                          for (int i=1; i < n; ++i)
                          {
                              int key = arr[i];
                              int j = i-1;
                   
                              /* Move elements of arr[0..i-1], that are
                                 greater than key, to one position ahead
                                 of their current position */
                              while (j>=0 && arr[j] > key)
                              {
                                  arr[j+1] = arr[j];
                                  j = j-1;
                              }
                              arr[j+1] = key;
                          }
                      }

Ejemplo

Lista x = [5,3,1,2,4]

inicia procesando el arreglo x, en cual en su posición 0 tiene el numero 5 este se compara con el 3 y se nota que el 5 es mayor que 3 por lo tanto se mueve a la derecha y el 3 pasa a la posición 0.

x = [3,5,1,2,4]

En la siguiente interacción esto se agrega el ítem numero 2 del arreglo (1) a la lista ordenada que contiene a 3 y 5, aquí se compara 1 con 5 y al ser mayor 5 se mueve a la derecha y se inserta el 1 en la posición 1 de la lista ordenada quedando [3,1,5], aquí se vuelve a comprar 1 con 3 y al ser mayor el 3 este se mueve a la derecha.

x = [1,3,5,2,4]

En la siguiente interacción se agrega el ítem numero 3 del arreglo a la lista ordenada quedando [1,3,5,2], aquí se comprar el 2 con el 5, y al ser mayor el 5 se mueve a la derecha quedando la lista [1,3,2,5], se vuelve a compara el 2 con el 3 y al ser mayor el 3 este se mueve la derecha quedando [1.2.3.5], se vuelve a comparar el 2 con el 1 y al ser mayor el 2 este se inserta en la posición 1 del arreglo y se termina la ejecución de esta interacción.

x = [1.2.3,5,4]

se realiza otra iteración y se agrega el ultimo ítem de la lista a la lista ordenada y se realiza la comparación con el 5, al ser mayor este se mueve a la derecha, se vuelve a comparar el 4 con el 3 y al ser mayor el 4 este se inserta en la posición 3 del arreglo y se termina la ejecución de esta interacción
